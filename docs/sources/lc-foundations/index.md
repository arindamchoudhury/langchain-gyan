# Foundation: Introduction to LangChain - Python

**Platform:** LangChain Academy
**URL:** https://academy.langchain.com/courses/take/foundation-introduction-to-langchain-python

Three-module course introducing LangChain's core features through building agents, from foundational patterns to production-ready systems.

## Modules

**Module 0 — Welcome**

- Course Overview (video)
- Getting Set Up
- Module 0 Resources

**Module 1 — Create Agent**

- Foundational models
- Tools
- Short-Term Memory
- Multimodal Messages
- Project: Personal Chef

**Module 2 — Advanced Agent**

- Model Context Protocol (MCP)
- Context and State
- Multi-Agent Systems
- Project: Wedding Planner

**Module 3 — Production-Ready Agent**

- What is Middleware?
- Managing Long Conversations
- Human In The Loop (HITL)
- Dynamic Agents
- Project: Email Assistant
- Bonus: Agent Chat UI
